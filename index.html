How To Play The Game ?

  • Objective: Clear a minefield without detonating any mines. The goal is to uncover all the safe squares.
  • Game Board: The board consists of a grid of squares, some of which contain mines. The size of the board and the number of mines can vary depending on the difficulty level (easy, medium, hard).
  • Starting the Game: Click on any square to start the game. The first click will always reveal a safe square.
  • Uncovering Squares: Clicking on a square will either uncover a number (indicating the count of adjacent mines) or a mine (which ends the game). If a square with no adjacent mines is clicked, it will automatically reveal all adjacent squares.
  • Numbers: Each uncovered number (1-8) indicates how many mines are in the surrounding squares (including diagonals). Use these numbers to deduce where mines might be located.
  • Flagging Mines: Right-click (or use a specific key combination) to place a flag on a suspected mine location. Flagging helps keep track of potential mines and prevents accidental clicks.
  • Winning the Game: The game is won when all non-mine squares are uncovered or all mines are flagged correctly.
  • Losing the Game: The game is lost if you click on a mine, which will end the game and reveal all mine locations.
  • Game Settings: Adjust settings for difficulty, including grid size and number of mines.
  • Strategy: Use logic and deduction to uncover squares and flag potential mines. Start by clicking on corners or edges where there are fewer adjacent squares.
